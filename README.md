# AI Fraud Detection with Geospatial Analytics

> Detect suspicious transactions using machine learning and location-based intelligence.

---

## Problem
Fraud detection systems often ignore spatial patterns, missing critical anomalies.

---

## Solution
This system combines:
- Machine Learning models
- Geospatial visualization
- Pattern detection

to identify fraud hotspots and anomalies.

---

## ⚙️ Features
- Fraud Classification Models
- Geospatial Mapping of Transactions
- Hotspot Detection
- Feature Engineering
- Real-time style analytics simulation

---

## System Flow
Transaction Data → Feature Engineering → ML Model →  
Geospatial Mapping → Fraud Insights
